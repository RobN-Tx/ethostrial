PI Web API 2015 R2 - Demo

The PI Web API is a RESTful service in the PI Developer Technologies suite, designed to provide cross-platform web and mobile programmatic interfaces to the PI System. The PI Web API presently contains basic functionality needed to retrieve and manipulate time series data from the PI Data Archive, and Asset and Event Frame data from the PI Asset Framework, to index and search on objects within the PI System.
